Objetivo da Aula

O objetivo desta aula é aprender a fazer requisições HTTP para um servidor usando o HttpClient do Angular.

Teoria com Analogia

Pense no HttpClient como um garçom em um restaurante. Seu componente (o cliente) faz um pedido (requisição HTTP) ao garçom. O garçom leva o pedido à cozinha (o servidor) e, depois de um tempo, traz a comida (a resposta do servidor).

Prática Apoiada

Vamos criar um serviço em Angular que busca uma lista de usuários de uma API de teste.


// user.service.ts
import { Injectable } from '@angular/core';
import { HttpClient } from '@angular/common/http';

@Injectable({ providedIn: 'root' })
export class UserService {
    constructor(private http: HttpClient) {}

    getUsers() {
        return this.http.get('https://jsonplaceholder.typicode.com/users');
    }
}

Perguntas de Múltipla Escolha

Qual módulo do Angular precisamos importar para usar o HttpClient?

A. FormsModule
B. HttpClientModule
C. RouterModule

Explicação: O `HttpClientModule` é essencial e precisa ser importado no módulo raiz da aplicação para que o serviço `HttpClient` esteja disponível para injeção de dependência.

Resumo

Nesta aula, aprendemos a usar o HttpClient do Angular para nos comunicarmos com APIs externas, um passo fundamental para construir aplicações ricas e dinâmicas.
