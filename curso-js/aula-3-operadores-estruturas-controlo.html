Aula 3: Operadores e Estruturas de Controlo

O que vai ser capaz de fazer

No final desta aula, conseguirá usar operadores para realizar cálculos e comparações, e utilizar estruturas de controlo como `if/else` e `for` para criar lógica e repetições no seu código.

Teoria (Com Analogia)

O código não é apenas uma lista de tarefas; ele precisa de tomar decisões e repetir ações. É aqui que entram os operadores e as estruturas de controlo.

Analogia da Receita de Cozinha Inteligente:

Imagine uma receita que se adapta sozinha:

  • Operadores: São as ações básicas da cozinha.
    • Aritméticos (`+`, `-`, `*`, `/`): Somar ou dividir ingredientes. Ex: `2 + 2`.
    • Comparação (`>`, `<`, `===`): Verificar se algo é verdade. Ex: "A temperatura do forno é superior a 180 graus?" (`temperatura > 180`).
    • Lógicos (`&&`, `||`): Combinar verificações. Ex: "Temos farinha E ovos?" (`temFarinha && temOvos`).
  • Estruturas de Controlo: São as instruções lógicas da receita.
    • `if/else` (Se/Senão): A tomada de decisão. "Se tivermos chocolate, adicionamos. Senão, usamos baunilha."
    • `for` (Para): A repetição. "Para cada um dos 10 biscoitos na bandeja, adicione uma amêndoa."

Prática (Guiada Passo a Passo)

Vamos criar um pequeno script que decide se um número é par ou ímpar e depois conta até esse número.

  1. Abra a consola do seu navegador (F12 ou Clique direito > Inspecionar > Consola).
  2. Declare uma variável e use `if/else`: 
    let numero = 10;

if (numero % 2 === 0) {
  console.log(numero + ' é um número par.');
} else {
  console.log(numero + ' é um número ímpar.');
}
// O operador '%' (módulo) dá o resto de uma divisão. Se o resto por 2 for 0, o número é par.
  3. Agora, adicione um loop `for` para contar de 1 até ao nosso número: 
    console.log('A contar até ' + numero + ':');

for (let i = 1; i <= numero; i++) {
  console.log(i);
}
// O loop começa com i=1; continua enquanto i for menor ou igual a 'numero'; e incrementa i a cada passo.
  4. Experimente mudar o valor da variável `numero` e veja como o resultado se altera.

Perguntas de Escolha Múltipla

1. Qual operador é usado para verificar se dois valores são estritamente iguais (valor e tipo)?




2. Que estrutura de controlo usaria para executar um bloco de código repetidamente, um número específico de vezes?




Resumo do que aprendeu

Nesta aula, aprendeu a dar "inteligência" ao seu código:

  • Operadores: Permitem-lhe manipular dados, desde simples somas (`+`) a comparações lógicas complexas (`===`, `&&`).
  • Estrutura `if/else`: É o pilar da tomada de decisões, permitindo que o seu código siga caminhos diferentes com base em condições.
  • Estrutura `for`: É a forma mais comum de criar repetições (loops), ideal para quando sabe exatamente quantas vezes precisa de repetir uma ação.
