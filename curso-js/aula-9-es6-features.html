Objetivo da Aula

O objetivo desta aula é conhecer as novas features introduzidas no ECMAScript 6 (ES6) e versões posteriores, como arrow functions, template literals e desestruturação.

Teoria com Analogia

Pense no JavaScript como uma caixa de ferramentas. O ES6 é como receber um upgrade nessa caixa, com ferramentas novas e mais eficientes que permitem fazer o mesmo trabalho de antes, mas de forma mais rápida e com menos esforço.

Prática Apoiada

Vamos reescrever um código antigo usando as novas features do ES6.


// Antes do ES6
var nome = "João";
console.log("Olá " + nome);

// Com ES6 (template literals)
const nome = "João";
console.log(`Olá ${nome}`);

Perguntas de Múltipla Escolha

Qual a principal vantagem das arrow functions?

A. São mais rápidas.
B. Têm uma sintaxe mais curta e um `this` léxico.
C. Funcionam em navegadores antigos.

Explicação: Arrow functions oferecem uma sintaxe mais concisa e herdam o `this` do contexto em que foram criadas, evitando confusões comuns com o `this` dinâmico das funções tradicionais.

Resumo

Nesta aula, exploramos as poderosas features do ES6+, como template literals e arrow functions, que nos permitem escrever um código JavaScript mais moderno e legível.
