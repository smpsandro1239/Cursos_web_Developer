Objetivo da Aula

O objetivo é aplicar os conhecimentos de JavaScript para adicionar interatividade à página de portfólio. Vamos criar um "dark mode", carregar projetos de uma API e manipular o DOM.

Teoria com Analogia

Se o HTML é a estrutura e o CSS é a decoração, o JavaScript é a eletricidade da casa. Ele permite que as coisas funcionem: acender e apagar luzes (dark mode), buscar informações de fora (API) e interagir com os moradores (eventos do usuário).

Prática Apoiada

O desafio é adicionar um botão que alterne a classe `dark-mode` no `<body>`. Além disso, use `fetch` para buscar dados de uma API (como a API do GitHub para listar seus repositórios) e exibi-los dinamicamente na seção de projetos.


// Exemplo para o dark mode
const btnToggle = document.getElementById('toggle-dark-mode');
btnToggle.addEventListener('click', () => {
    document.body.classList.toggle('dark-mode');
});

// Exemplo para buscar dados da API
fetch('https://api.github.com/users/SEU-USUARIO/repos')
    .then(response => response.json())
    .then(data => {
        // Lógica para exibir os repositórios na página
    });

Perguntas de Múltipla Escolha

Qual é a maneira moderna e mais comum de fazer requisições assíncronas em JavaScript?

A. Usando `XMLHttpRequest`.
B. Usando a `fetch` API com Promises ou `async/await`.
C. Usando jQuery `$.ajax`.

Explicação: A `fetch` API é a interface nativa do navegador para fazer requisições de rede. Ela é baseada em Promises, tornando o código assíncrono mais limpo e legível, especialmente com `async/await`.
