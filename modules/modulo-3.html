Listas e Links

Listas organizam conteúdo em sequência e links criam navegacção entre páginas e seções. São elementos fundamentais para boa usabilidade.

🎯 O que vamos aprender:

  • Listas aninhadas (menus)

  • Âncoras e navegacção

Listas Ordenadas

As listas ordenadas (

    ) criam listas numeradas. São ideais quando a ordem dos itens é importante.

    📝 Como usar:

      
    
    1. Primeiro passo
      2. 
    
    2. Segundo passo
      3. 
    
    3. Terceiro passo
      4.

    🎯 Quando usar
      :

    • Instruções passo-a-passo
    • Rankings ou top lists
    • Sequências cronológicas
    • Processos ordenados

Exemplo: Receita de Pastel de Nata

  1. Aquecer o forno a 250°C
  2. Preparar o recheio com leite e gemas
  3. Estender a massa folhada
  4. Rechear e levar ao forno

Links Básicos

Links () conectam páginas e recursos. São a essência da navegacção web.

🔗 Tipos de links:

  1. Externos: Para outros sites
  2. Internos: Para páginas do mesmo site
  3. Downloads: Para arquivos
  4. Email: Para composição de email

📝 Exemplos:

Google
Sobre Nós
Baixar PDF
Email
🌐 Site Externo https://exemplo.com
📄 Página Interna sobre.html
📎 Download arquivo.pdf
✉️ Email contacto@site.com

Target="_blank" e Âncoras

O atributo target="_blank" abre links em nova aba. Âncoras criam links internos que "saltam" para seções específicas da página.

📝 Nova aba e âncoras:


Novo site


Ir para Contatos
...

Seção de Contatos

🎯 Melhores práticas:

  • Use target="_blank" para sites externos
  • Sempre adicione rel="noopener noreferrer" por segurança
  • Nomes de ID devem ser únicos
  • Use âncoras para navegacção longa

Exercício Prático

Vamos criar uma página com índice navegável que salta para diferentes seções!

🎯 Tarefa:

Crie uma página com 3 seções principais e um índice no topo com links âncora.

📋 Requisitos:

  1. Estrutura HTML5 completa
  2. Índice de navegacção no topo
  3. 3 seções com IDs únicos
  4. Links âncora funcionais
  5. Listas para organizar conteúdo
  6. Pelo menos um link externo
