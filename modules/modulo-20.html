Design Responsivo

O site deve funcionar perfeitamente em desktop, tablet e mobile.

📱 Meta viewport:

🎨 CSS responsivo:

/* Mobile first */
.container {
    width: 100%;
    padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        max-width: 768px;
        margin: 0 auto;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        max-width: 1024px;
    }
}

Deploy GitHub Pages

Publique seu site gratuitamente no GitHub Pages para o mundo ver.

🚀 Passos para deploy:

  1. Criar repositório no GitHub
  2. Fazer push do código
  3. Ativar GitHub Pages nas settings
  4. Escolher branch e pasta (geralmente /root)
  5. Aguardar deploy automático

📝 Comandos Git:

git init
git add .
git commit -m "Projeto final: Agência de Viagens"
git branch -M main
git remote add origin https://github.com/usuario/agencia-viagens.git
git push -u origin main
1 Push to GitHub
2 GitHub Pages ativa
3 Site live!
